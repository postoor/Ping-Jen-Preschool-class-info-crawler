## 使用方式

1. 登入後從瀏覽器複製cookie到`.env`，格式參考`env.example`
2. (可選) 填入n8n的webhook連結
3. (可選) 複製system file設定timer
4. `pip install -r requirements.txt`
5. 執行`python main.py`
